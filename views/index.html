Explore los tipos de certificados que ofrecemos y seleccione el que se ajuste a sus requerimientos:

PAdES

Firma electrónica avanzada para documentos PDF.

PAdES proporciona un formato avanzado para firmas electrónicas en documentos PDF, asegurando la validez a largo plazo, incluso si los estándares tecnológicos cambian.

CAdES

Firma electrónica avanzada para documentos electrónicos.

CAdES extiende el formato de firma digital CMS para ofrecer firmas electrónicas avanzadas, compatibles con requerimientos legales y de seguridad a largo plazo.

XAdES

Firma electrónica para documentos XML.

XAdES ofrece un estándar para firmas electrónicas en documentos XML, garantizando integridad, autenticidad y validez a largo plazo.

Hash

Función criptográfica para la seguridad de datos.

Las funciones Hash transforman datos de cualquier tamaño en una salida de tamaño fijo, esencial para la integridad y la seguridad en la criptografía.

OTP

Contraseñas de un solo uso para seguridad mejorada.

OTP genera contraseñas únicas y de un solo uso para autenticaciones seguras, reduciendo significativamente el riesgo de accesos no autorizados.
